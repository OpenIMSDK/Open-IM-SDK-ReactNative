OpenIM SDK for react-native.

## Installation

```sh
npm install react-native-open-im-sdk
```

## Example

for ios:
  - git clone
  - yarn
  - connect real iphone
  - open `example/ios/OpenImSdkExample.xcworkspace` by `xcode`
  - build

## Usage

```js
import openIM from "react-native-open-im-sdk";

// ...

openIM.initSDK()
```
